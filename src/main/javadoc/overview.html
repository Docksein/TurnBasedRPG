Turn-Based RPG Game

A Java-based text adventure RPG with combat, exploration, and inventory management

Overview

This is a turn-based RPG game implemented in Java that combines room-based dungeon crawling with combat and inventory management. The player navigates through interconnected rooms, encounters various enemies, collects items, and progresses through levels.

Core Components

Entity System

  • Entity - Abstract base class for all game characters
  • Player - Represents the player character with XP, leveling, and inventory
  • Enemy - Represents hostile creatures with different types (common/boss)

Environment

  • Map - Class that loads map data from a text file as a hashtable of rooms
  • Room - Represents a single room with name, description and optional directions to other rooms

Item System

  • Item - Abstract base class for all usable items
  • Weapon - Equipable item that increases player damage
  • HealthPotion - Consumable items that restore health
  • StrengthPotion - Temporary damage boosters for combat encounters
  • Inventory - Container system with weapon slot and item storage

Game Systems

  • Combat - Turn-based battle system with multiple player actions
  • EnemyGenerator - Enemy creation based on player level and room name
  • ItemGenerator - Random item generation with level scaling

User Interface

  • InputHandler - Handles all user input and menu displays
  • Main - Game loop and state management

Game Flow

  1. Room Navigation - Player moves between connected rooms
  2. Enemy Encounters - Enemies spawn in unvisited rooms, bosses in designated boss rooms
  3. Combat Resolution - Turn-based combat with attack, guard, item use, and flee options
  4. Exploration - Players can search rooms for hidden items
  5. Character Progression - XP gain leads to level increases and stat improvements
  6. Inventory Management - Collect, equip, use, and discard items
